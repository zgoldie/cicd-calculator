# CI/CD Pricing Calculator

Compare estimated monthly costs for **Codemagic**, **GitHub Actions**, and **Bitrise**.

## For sales

1. Open `pricing-calculator.html` in Chrome, Safari, or Edge (double-click the file).
2. Set team size, build volume, and machine types.
3. Toggle which providers to compare.
4. Click **Calculate costs**.

No install, login, or hosting required. Works offline. Share the HTML file as-is (email, Drive, Slack, etc.).

## Updating pricing

Edit the constants at the top of the `<script>` block in `pricing-calculator.html` (`RATES`, `CM_PLANS`, `BITRISE_PRO_TIERS`, etc.).
